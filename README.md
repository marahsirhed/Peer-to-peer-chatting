# Peer-to-Peer Chatting Application

A Java-based Peer-to-Peer chatting application that allows users to communicate directly using UDP sockets, while a TCP server is used to manage login, active users, and user statuses. The application supports sending and receiving messages, broadcasting to online users, archiving and restoring deleted messages, exporting chat history, logging activities, and transferring files between peers with transfer statistics such as file size, packet count, transfer time, transfer rate, average delay, and jitter.

The project was built using Java and JavaFX for the graphical user interface. It uses UDP Socket Programming for direct peer-to-peer messaging, TCP Socket Programming for server-client communication and active user management, and Java File I/O for saving logs, exporting chat history, managing user credentials, and handling file transfer operations.
